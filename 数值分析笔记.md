### 1. 计算一阶和二阶均差（差商）

已知三个数据点：

- $(x_0, y_0) = (1.10, 1.21)$
    
- $(x_1, y_1) = (1.20, 1.32)$
    
- $(x_2, y_2) = (1.30, 1.47)$
    

**计算一阶均差：**

$$f[x_0, x_1] = \frac{y_1 - y_0}{x_1 - x_0} = \frac{1.32 - 1.21}{1.20 - 1.10} = \frac{0.11}{0.10} = 1.1$$

$$f[x_1, x_2] = \frac{y_2 - y_1}{x_2 - x_1} = \frac{1.47 - 1.32}{1.30 - 1.20} = \frac{0.15}{0.10} = 1.5$$

**计算二阶均差：**

$$f[x_0, x_1, x_2] = \frac{f[x_1, x_2] - f[x_0, x_1]}{x_2 - x_0} = \frac{1.5 - 1.1}{1.30 - 1.10} = \frac{0.4}{0.20} = 2.0$$

### 2. 构造牛顿二次插值多项式

根据牛顿插值公式，二次插值多项式为：

$$P_2(x) = f(x_0) + f[x_0, x_1](x - x_0) + f[x_0, x_1, x_2](x - x_0)(x - x_1)$$

将已知数值和计算出的均差代入：

$$P_2(x) = 1.21 + 1.1(x - 1.10) + 2.0(x - 1.10)(x - 1.20)$$

### 3. 展开并化简为 $ax^2 + bx + c$ 形式

现在我们需要将上面的式子展开合并同类项：

首先展开最后一项的乘积：

$$(x - 1.10)(x - 1.20) = x^2 - 1.2x - 1.1x + 1.32 = x^2 - 2.3x + 1.32$$

将其代回原式：

$$P_2(x) = 1.21 + 1.1x - 1.21 + 2.0(x^2 - 2.3x + 1.32)$$

继续展开括号：

$$P_2(x) = 1.1x + 2.0x^2 - 4.6x + 2.64$$

合并 $x$ 的同类项：

$$P_2(x) = 2.0x^2 + (1.1 - 4.6)x + 2.64$$

$$P_2(x) = 2.0x^2 - 3.5x + 2.64$$

### 4. 最终答案（应用精度要求）

根据试卷顶部“计算结果保留四位小数”的要求，我们需要对系数进行补零。

最终的二次插值多项式为：

$$P_2(x) = 2.0000x^2 - 3.5000x + 2.6400$$
